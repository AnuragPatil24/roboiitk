__Week 1: “Hello World\!” — or How to Talk to Robots Without Sounding Awkward__

#### <a id="_3s0jbkblqyhi"></a>__Objective:__ Get comfy with C\+\+ and Python, learn to tame the electrons, and impress robots enough to make them obey your commands\.

1. __C\+\+__
	- [Learn C\+\+ \(It’s not just for wizards\)](https://www.learncpp.com/)
	- Recommended Reading: Start with Chapter 1 \(Basics\) — because you can't speak robot if you don’t know your “cout” from your “cin”\. Stop at Chapter 3 once you've mastered making loops dance\.
2. __ Python:__
	- [Python Crash Course by Eric Matthes \(aka The Art of Talking to Snakes\)](https://khwarizmi.org/wp-content/uploads/2021/04/Eric_Matthes_Python_Crash_Course_A_Hands.pdf)
	- Suggested Slytherins: Skim through Chapters 1–3 and get cosy with variables, loops, and—yes, believe it or not—indentation\.
	- [W3Schools \(Where the Python Elders Live\)](https://www.w3schools.com/python/)
	- Chapters to Tackle: Basics, Strings, and Control Structures \(the “no ifs, ands, or buts” of Python\)\.
3. __ Introduction to Electronics:__
	- Brush up on voltage, current, and resistance—the holy trinity of electronic sorcery\. Try SparkFun’s Handy Tutorials for an easy\-to\-digest start\.

__Assignments:__

1. __C\+\+:__
	- Write a program that makes the computer do some maths for you—like a lazy student calculating addition, subtraction, multiplication, and division\.
	- Implement a “Guess the Number” game: the computer picks a number, and you guess it\. \(Don’t worry—it’s not sentient\. Yet\.\)
2. __Python:__
	- Build a script that calculates the factorial of a number using recursion \(yes, this is your first taste of *inception*\)\.
	- Go  through Level 1 questions from  
	-  [100\+ Python challenging programming exercises for Python 3\.md](https://github.com/zhiwehu/Python-programming-exercises/blob/master/100%2B%20Python%20challenging%20programming%20exercises%20for%20Python%203.md)  

### <a id="_o05n9f8x9izw"></a>__Week 2: Git, GitHub, Terminal__

#### <a id="_f2jrs6g7zhst"></a>__Objective: __Master the essentials of version control with Git and GitHub, and become comfortable navigating through the Linux terminal\.

__Resources:__

1. __Introduction to Git & GitHub:__
	- [Git & GitHub Crash Course](https://www.youtube.com/watch?v=tRZGeaHPoaw&ab_channel=KevinStratvert)
2. __Basic Terminal Commands:__
	- Understanding basic commands to access and manipulate the Linux operating system from the terminal is essential\.
	- Example Commands: cd, ls, pwd, mkdir, rm, cp, mv, nano, and more\.
	- Reference:[ Linux Terminal Basics](https://homepages.uc.edu/~thomam/Intro_Unix_Text/File_System.html)
3. __Git Basics:__
	- [Git Basics Article](https://medium.com/@tom.puntillo/what-is-git-c150dee640a)
	- Overview of core concepts, including version control, branches, merges, and collaboration\.
	- [Git Documentation](https://git-scm.com/book/en/v2)
	- Dive deeper into the Git documentation to learn commands like git init, git commit, git push, and how to resolve merge conflicts\.
4. __Linux Fundamentals:__
	- [Understanding Linux Filesystem](https://homepages.uc.edu/~thomam/Intro_Unix_Text/File_System.html)
	- Linux Crash Course \- [Linux Crash Course \- YouTube](https://www.youtube.com/playlist?list=PLT98CRl2KxKHKd_tH3ssq0HPrThx2hESW)
	- When working with Linux, it’s crucial to understand the filesystem and hierarchy tree\.

__Assignments:__

- 
	- Complete the [OverTheWire Bandit](https://overthewire.org/wargames/bandit/) game till Level 15:  
This interactive game teaches essential Linux commands and file system navigation skills\.
